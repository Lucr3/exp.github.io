How This Data Was Used

Global datasets were cleaned and processed through Python to ensure accuracy and consistency in the interactive charts presented.

Methodology

Data Cleaning

What We Did

We kept the data as original as possible. Starting from large global datasets, we simply extracted only the rows concerning Yemen, discarding everything else. We did not modify the numbers or artificially fill in missing data: if you see "gaps" in the charts, it's because that data was genuinely missing from the source.

Data Processing

How We Went From Numbers to Charts

All the work happens directly in your browser. The code reads the files and uses the D3.js library to transform numbers into visual elements:

  • GPS coordinates become points and areas on the Yemen map.
  • Dates and numbers are converted into lines, spirals, or colored bars to make the timeline of the crisis immediately comprehensible.

Limitations

What You Should Know Before Reading the Data

Collecting data in a war zone is very difficult, so keep in mind that:

  • Missing data: For some years (especially for education) there is no data because institutions were not functioning.
  • Isolated zones: Conflict reports may not capture everything that happens in the most remote and hard-to-reach areas.
  • Perspective: Google Trends data shows us only what people search for on the internet, not the actual severity of what happens on the ground.
